Hospital Readmission Analysis – Excel Project

1. Project Overview

This Excel project analyzes hospital readmission data to identify key factors affecting patient readmission rates. The objective is to evaluate demographic, clinical, and visit-related patterns and present insights through an interactive dashboard.

2. Tools Used
   
Microsoft Excel
Pivot Tables & Pivot Charts
Slicers
Calculated Fields
Conditional Formatting

3. Key KPIs

Overall Readmission Rate

Average Length of Stay

Diabetic Readmission Rate

Average Emergency Visits per Patient

4. Key Insights

Patients aged 80–90 years have the highest readmission rate.

Caucasian females (47.75%) represent a significant portion of readmissions.

Patients with frequent emergency visits are more likely to be readmitted.

Average lab tests per patient: 43, with higher counts linked to increased readmission.

Readmission is slightly higher (48.62%) among patients whose diabetic medication was changed.

Out of 51,205 diabetic patients, diabetic individuals show higher readmission tendency.

Increased outpatient follow-ups reduce readmission risk.

5. Dashboard Features
 
The dashboard includes dynamic visualizations for:

Readmission by Age Group

Race & Gender Distribution

Average Hospital Stay by Medical Specialty

Emergency Visit Impact on Readmission

Outpatient Visits vs Readmission Trend

Diabetic Readmission Rate Analysis

Emergency & Outpatient Visit Impact

6. Conclusion
 
The analysis identifies elderly patients, diabetic individuals, and high emergency visit frequency as major drivers of readmission. Preventive care through increased outpatient follow-ups can help reduce readmission rates.

The interactive Excel dashboard enables quick identification of high-risk segments and supports data-driven healthcare decisions.
